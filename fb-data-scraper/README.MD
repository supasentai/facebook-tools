### How to use Data Scraper (Comment Extractor)

1. Open the specific Facebook post you want to extract data from.
2. Click "View more comments" to expand and display all the comments you want to capture.
3. Open **Developer Tools** (`F12`) -> **Console** tab.
4. Copy the entire contents of `fb-data-scraper.js`, paste it into the console, and press **Enter**.
5. A `.csv` file containing the structured data will automatically download to your machine.
