- **Friend Request Sender (`fb-send-request.js`)**: Automatically sends friend requests to multiple users on the "People You May Know" or specific friend lists with adjustable safe delays.

### How to use Friend Request Sender

1. Navigate to the Facebook Friends recommendation page:
   `https://www.facebook.com/friends/suggestions` (or any profile's friend list).
2. Scroll down to load enough users you want to send requests to.
3. Open **Developer Tools** (`F12`) -> **Console** tab.
4. Copy the entire contents of `fb-send-request.js`, paste it into the console, and press **Enter**.
